# 📱 Google Play 앱 카테고리 변경 가이드

## 📋 개요

소셜/데이트 카테고리에서 다른 카테고리로 변경하여 조직 계정 없이 앱을 배포할 수 있습니다.

---

## 🎯 권장 카테고리

### 1. 라이프스타일 (Lifestyle) - **가장 추천**

**장점**:
- 조직 계정 불필요
- 소셜/데이트 앱과 유사한 기능 허용
- 광범위한 사용자 타겟팅 가능

**적합한 이유**:
- 사람 간 연결 서비스
- 프로필 기반 매칭
- 커뮤니티 기능

### 2. 엔터테인먼트 (Entertainment)

**장점**:
- 조직 계정 불필요
- 재미와 오락 중심 서비스

**적합한 이유**:
- 매칭 게임 요소
- 커뮤니티 활동

### 3. 생산성 (Productivity)

**장점**:
- 조직 계정 불필요
- 실용적 도구로 포지셔닝 가능

**단점**:
- 소셜 기능과 거리가 멀 수 있음

---

## 📝 단계별 변경 방법

### 1단계: Google Play Console 접속

1. [Google Play Console](https://play.google.com/console/) 접속
2. 앱 선택

### 2단계: 앱 카테고리 변경

1. 왼쪽 메뉴에서 **앱 콘텐츠** (App content) 클릭
2. **앱 카테고리** (App category) 섹션 찾기
3. **주 카테고리** (Primary category) 변경:
   - 현재: **소셜** (Social) 또는 **데이트** (Dating)
   - 변경: **라이프스타일** (Lifestyle) 또는 **엔터테인먼트** (Entertainment)
4. **저장** 클릭

### 3단계: 앱 설명 수정

카테고리에 맞게 앱 설명을 조정해야 할 수 있습니다:

#### 변경 전 (소셜/데이트)
```
"어트랙트는 소개팅 앱입니다..."
"이성과의 만남을 위한 서비스..."
```

#### 변경 후 (라이프스타일)
```
"어트랙트는 새로운 사람들과 만나고 연결하는 라이프스타일 앱입니다..."
"관심사가 비슷한 사람들과의 네트워킹 서비스..."
"일상 속에서 새로운 인연을 만나는 플랫폼..."
```

### 4단계: 앱 기능 선언 확인

1. **앱 콘텐츠** → **앱 액세스** → **앱 기능** (App access → App functionality)
2. 다음 항목 확인:
   - **사용자 생성 콘텐츠**: "예" (UGC 허용)
   - **소셜 기능**: 필요시 조정
   - **데이트 기능**: 제거하거나 완화

### 5단계: 타겟 고객층 확인

1. **정책** → **앱 콘텐츠** → **타겟 고객층**
2. 연령 제한 확인:
   - 만 18세 이상: 유지
   - 만 13세 이상: 변경 고려 (더 넓은 타겟)

## ⚠️ 주의사항

### 1. 기능 선언 일관성

앱 카테고리를 변경했지만 앱 기능 선언에서 여전히 "데이트 앱"이라고 명시하면 문제가 될 수 있습니다.

**확인할 항목**:
- 앱 기능 선언
- 앱 설명
- 스크린샷
- 마케팅 자료

### 2. 키워드 조정

앱 설명에서 사용하는 키워드를 조정:

**제거/완화할 키워드**:
- "소개팅"
- "데이트"
- "이성 만남"
- "연인 찾기"

**대체 키워드**:
- "네트워킹"
- "인연 만들기"
- "새로운 사람 만나기"
- "커뮤니티"

### 3. 스크린샷 업데이트

스크린샷에서도 데이트 관련 문구가 있다면 수정:

- "소개팅" → "매칭" 또는 "연결"
- "이성 만남" → "새로운 인연"
- "데이트" → "만남" 또는 "활동"

## 🔄 변경 후 확인 사항

### 1. 정책 준수 확인

1. **정책 상태** (Policy status) 확인
2. 경고나 오류 메시지 확인
3. 모든 섹션 완료 확인

### 2. 앱 제출

1. **프로덕션** (Production) 또는 **내부 테스트** (Internal testing) 선택
2. **새 버전 만들기** 클릭
3. 앱 제출
