# 📢 카카오 애드핏 광고 연동 가이드

## 📋 현재 상태

- ✅ `KakaoAdFit` 컴포넌트 구현 완료
- ✅ 더미 배너 표시 중 (실제 광고 대기)
- ⏳ 카카오 애드핏 계정 및 광고 단위 생성 필요

---

## 🚀 단계별 설정 가이드

### 1단계: 카카오 애드핏 가입 및 사이트 등록

#### 1-1. 카카오 애드핏 가입

1. **카카오 애드핏 접속**
   - https://adfit.kakao.com 접속
   - 카카오 계정으로 로그인

2. **사이트 등록**
   - 대시보드 > 사이트 관리 > 사이트 등록
   - 사이트 정보 입력:
     - 사이트명: `Attract` (또는 원하는 이름)
     - 사이트 URL: `https://attract.web.app` (또는 실제 도메인)
     - 사이트 카테고리: `소셜/커뮤니티` 또는 `데이팅/만남`
     - 사이트 설명: 서비스 소개

3. **사이트 승인 대기**
   - 보통 1-3일 소요
   - 승인 완료 시 이메일 알림

---

### 2단계: 광고 단위 생성

#### 2-1. 배너 광고 단위 생성

1. **광고 단위 생성**
   - 대시보드 > 광고 단위 관리 > 광고 단위 생성
   - 광고 형식: `배너 광고` 선택

2. **광고 단위 설정**
   - 광고 단위명: `홈 화면 상단 배너` (또는 원하는 이름)
   - 광고 크기: `320x100` (또는 원하는 크기)
   - 광고 위치: `홈 화면 상단`
   - 광고 단위 ID 복사: `DAN-XXXXXXXXXX` 형식

3. **추가 광고 단위 생성** (필요한 만큼)
   - 예: `홈 화면 하단 배너`, `프로필 페이지 배너` 등

---

### 3단계: 환경 변수 설정

#### 3-1. `.env` 파일에 광고 단위 ID 추가

```env
# 카카오 애드핏 광고 단위 ID
VITE_KAKAO_AD_UNIT_1=DAN-XXXXXXXXXX
VITE_KAKAO_AD_UNIT_2=DAN-YYYYYYYYYY
```

#### 3-2. 환경 변수 사용 (선택사항)

여러 광고 단위를 관리하기 쉽게 환경 변수로 관리할 수 있습니다.

---

### 4단계: 코드에 광고 단위 ID 적용

#### 4-1. 환경 변수에서 가져오기 (권장)

광고 단위 ID를 환경 변수로 관리하면 프로덕션/개발 환경을 쉽게 전환할 수 있습니다.

#### 4-2. 직접 입력하기

코드에 직접 광고 단위 ID를 입력할 수도 있습니다.

---

## 📝 코드 적용 예시

### 방법 1: 환경 변수 사용 (권장)

```tsx
// src/pages/home/components/ProfileTab.tsx
import KakaoAdFit from '../../../components/ads/KakaoAdFit';

// 환경 변수에서 가져오기
const AD_UNIT_1 = import.meta.env.VITE_KAKAO_AD_UNIT_1 || '';
const AD_UNIT_2 = import.meta.env.VITE_KAKAO_AD_UNIT_2 || '';

// 사용
{AD_UNIT_1 ? (
  <KakaoAdFit unit={AD_UNIT_1} width={320} height={100} />
) : (
  <DummyAdBanner width={320} height={100} text="광고 영역 1" />
)}
```

### 방법 2: 직접 입력

```tsx
// src/pages/home/components/ProfileTab.tsx
import KakaoAdFit from '../../../components/ads/KakaoAdFit';

// 직접 광고 단위 ID 입력
<KakaoAdFit unit="DAN-XXXXXXXXXX" width={320} height={100} />
```

## 🔧 광고 컴포넌트 개선

현재 `KakaoAdFit` 컴포넌트를 개선하여 광고가 로드되면 자동으로 표시되도록 할 수 있습니다.

---

## 📊 광고 수익 최적화 팁

1. **광고 위치**
   - 사용자가 자주 보는 위치에 배치
   - 콘텐츠 사이에 자연스럽게 배치

2. **광고 크기**
   - 모바일: 320x100 (권장)
   - 데스크톱: 728x90 또는 320x100

3. **광고 개수**
   - 너무 많으면 사용자 경험 저하
   - 적절한 개수 유지 (페이지당 2-3개 권장)

4. **광고 리프레시**
   - 카카오 애드핏은 자동으로 광고를 갱신합니다
   - 수동 리프레시는 권장하지 않습니다

---

## 🧪 테스트

### 개발 환경 테스트

1. `.env` 파일에 테스트 광고 단위 ID 설정
2. 개발 서버 실행: `npm run dev`
3. 광고가 표시되는지 확인

### 프로덕션 배포

1. 프로덕션 환경 변수 설정
2. 빌드: `npm run build`
3. 배포: `firebase deploy`
4. 실제 사이트에서 광고 확인

---

## ⚠️ 주의사항

1. **사이트 승인 필요**
   - 카카오 애드핏은 사이트 승인 후 광고가 표시됩니다
   - 승인 전에는 광고가 표시되지 않습니다

2. **광고 정책 준수**
   - 카카오 애드핏 정책을 준수해야 합니다
   - 부적절한 콘텐츠가 있으면 승인이 거부될 수 있습니다

3. **광고 단위 ID 보안**
   - 광고 단위 ID는 공개되어도 안전합니다
   - 하지만 환경 변수로 관리하는 것을 권장합니다
