# 📊 광고 표시 상태 확인 가이드

## 🔍 현재 광고 표시 로직

코드에서 광고는 다음과 같이 표시됩니다:

```tsx
// 환경 변수 확인
const KAKAO_AD_UNIT_1 = import.meta.env.VITE_KAKAO_AD_UNIT_1 || '';

// 조건부 렌더링
{KAKAO_AD_UNIT_1 ? (
  <KakaoAdFit unit={KAKAO_AD_UNIT_1} width={320} height={100} />
) : (
  <DummyAdBanner width={320} height={100} text="광고 영역 1" />
)}
```

---

## ✅ 현재 표시되는 것

### 경우 1: 환경 변수가 설정되어 있는 경우

**표시되는 것**: 실제 카카오 애드핏 광고
- `.env` 파일에 `VITE_KAKAO_AD_UNIT_1=DAN-xzcTogwnUXAFXJ7t`가 있으면
- `KakaoAdFit` 컴포넌트가 렌더링됨
- 카카오 애드핏 스크립트가 로드되고 실제 광고가 표시됨

**확인 방법**:
1. 브라우저 개발자 도구(F12) → Elements 탭
2. `<ins class="kakao_ad_area">` 요소 확인
3. `data-ad-unit="DAN-xzcTogwnUXAFXJ7t"` 속성 확인

### 경우 2: 환경 변수가 없는 경우

**표시되는 것**: 더미 배너 (회색 점선 박스)
- `.env` 파일에 광고 단위 ID가 없으면
- `DummyAdBanner` 컴포넌트가 렌더링됨
- "광고 영역 1", "광고 영역 2" 텍스트가 있는 회색 박스 표시

**확인 방법**:
- 화면에 회색 점선 박스와 "광고 영역" 텍스트가 보이면 더미 배너

---

## 🔍 현재 상태 확인 방법

### 1. 브라우저 콘솔에서 확인

1. 개발 서버 실행: `npm run dev`
2. 브라우저에서 홈 화면 접속
3. F12 → Console 탭 열기
4. 다음 명령어 실행:

```javascript
// 환경 변수 확인
console.log('광고 단위 ID:', import.meta.env.VITE_KAKAO_AD_UNIT_1);

// 광고 요소 확인
console.log('카카오 애드핏 요소:', document.querySelector('.kakao_ad_area'));
```

### 2. Elements 탭에서 확인

1. F12 → Elements 탭
2. `<ins class="kakao_ad_area">` 요소 찾기
3. `data-ad-unit` 속성 확인:
   - 있으면: 실제 광고 (환경 변수 설정됨)
   - 없으면: 더미 배너 (환경 변수 없음)

### 3. 시각적으로 확인

**실제 광고가 표시되는 경우**:
- 카카오 애드핏 광고 콘텐츠가 표시됨
- 광고 이미지나 텍스트가 보임

**더미 배너가 표시되는 경우**:
- 회색 점선 박스
- "광고 영역 1", "광고 영역 2" 텍스트
- 광고 아이콘 (ri-advertisement-line)

---

## 🚨 광고가 보이지 않는 경우

### 1. 환경 변수 미설정

**증상**: 더미 배너가 표시됨

**해결**:
1. `.env` 파일 생성 또는 수정
2. `VITE_KAKAO_AD_UNIT_1=DAN-xzcTogwnUXAFXJ7t` 추가
3. 개발 서버 재시작

### 2. 카카오 애드핏 사이트 미승인

**증상**: 실제 광고 요소는 있지만 광고 콘텐츠가 표시되지 않음

**해결**:
1. 카카오 애드핏 대시보드에서 사이트 승인 상태 확인
2. 승인 대기 중이면 승인 완료까지 대기 (1-3일)

### 3. 스크립트 로드 실패

**증상**: 브라우저 콘솔에 에러 메시지

**확인**:
- F12 → Console 탭에서 에러 확인
- 네트워크 탭에서 `ba.min.js` 로드 확인
